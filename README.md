# GymLens Backend

This repository contains the backend implementation for the GymLens app, which is an application designed to categorize gym equipment and provide users with detailed information, and guidance on proper usage.

## Project Description

The GymLens backend is built using the Go programming language and the Fiber framework. It provides API endpoints for user authentication and user-related operations. The backend utilizes JWT for authentication, Firestore, and Cloud Storage for data storage.
